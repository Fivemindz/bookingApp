Title: 
  Serverless Booking Application

Description: 
  This project will allow a user to create, retrieve, update and delete bookings(appointments). A user can also upload a picture.

How to use:
  This project is deployed and ready for testing using the included postman collection. An authorization token is already saved in the postman collection.

Endpoints:
  getBookings - get bookings for user using authorization token
  createBooking - allows user to book appointment
  updateBooking - updates booking confirmation
  deleteBooking - deletes booking using the booking included
  uploadAttachment - allows user to upload an attachment
